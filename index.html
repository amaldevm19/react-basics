React JS - A Javascript front-end framework

  • Download and install Node Js from https://nodejs.org
  • NPM is automatically installed along with Node JS https://www.npmjs.com/package/react
  • Download and install Visual Studio Code from https://code.visualstudio.com
  • Create a workfolder
  • Open workfolder in CMD/Power shell
  • To create a react app
  • https://reactjs.org/ 
    •             npx create-react-app my_app --use-npm
  • Change your directory to new app folder
    •             cd my_app
  • Open react project in your code editor
    •             code .
  • To start react app run
    •             npm start
  • Check your local host for running react app
    •            http://localhost:3000/
  • Go to VScode and do modifications in App.js
